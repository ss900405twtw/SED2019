\begin{itemize}
    \item Assign codes to team members
    \item Review UML tools for Python
    \begin{itemize}
        \item Pynsource
        \begin{itemize}
            \item Easy to use with GUI
            \item Fails to edit the class diagram in community edition
            \item Fails to track the dependency across multiple files
        \end{itemize}
        
        \item \href{https://hackmd.io/FBodxFWQR4uSB1kP0sMwQQ?fbclid=IwAR26S-VNRVV-t2_Op1JFY3yCBzePwmPSrm7k5v-uSRmXWNjQ3MFBf6hNj48}{Pyreverse}
        \begin{itemize}
            \item Free and open-source.
            \item Fails to capture some dependencies (e.g. when class A stores instances of class B in a list instead of directly holding the reference). 
            \item Fails to generate complete diagram for some files (with errors)
            \item Another open source tool, graphviz2drawio, can convert the .dot file generated by Pyreverse to a format editable by draw.io, but it does not work.
        \end{itemize}
        \item\href{https://drive.google.com/file/d/1pAtm75X_akqSj1yn7SvmzXEujhWLID0S/view?usp=sharing}{Start from scratch on draw.io}
        \begin{itemize}
            \item draw class diagram by hand
            \item very time-consuming (took almost 20 hrs to finish reading and drawing class diagram for a 1000 lines python file)
            \item \includegraphics[scale=0.6]{draw_io}
        \end{itemize}
    \end{itemize}
    \item \href{https://docs.google.com/presentation/d/1eeXQm_IOWkrKw5gyJf5k0Bbf7E3tWRf51N2ut9YpDV8/edit?usp=sharing}{Code reading progress}
\end{itemize}\\
